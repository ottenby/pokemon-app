Pokemon-App

Description:
Pokemon-App is a web application that allows users to browse and explore a list of Pokémon. Users can view detailed information about each Pokémon, including their stats, type, and more.

Installation:
To get started and set up the project, run the following command in the terminal:
npm install

Usage:

To run the project locally, use the command:
npm run dev

To build the project, use:
npm run build

For testing the project, run:
npm run test

Technologies Used

    •	Next.js v16
    •	TypeScript
    •	Tailwind CSS
    •	Axios
    •	Jest
